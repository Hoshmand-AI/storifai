How it works

A deep learning pipeline that reads your photos and writes a connected narrative.

Step 01
Upload your photos
Select 3–5 photos from any event — travel, family, everyday moments.
Step 02
Vision encoding
ResNet-50 extracts visual features from each photo — objects, scenes, colors, textures.
Step 03
Story generation
An LSTM decoder reads the image features and generates a sentence-by-sentence story.
Step 04
Copy and share
Copy your story and post it to Instagram, a travel blog, or share with family.

About this model

Built as the baseline model for the Storifai deep learning research project.

This is the baseline version of Storifai, built using a ResNet-50 image encoder and LSTM decoder trained on the VIST dataset (40,155 human-written photo stories). The improved version uses CLIP encoding and cross-image attention for richer, more coherent narratives.
